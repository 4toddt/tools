# tools

A grab-bag of small browser tools and prototypes I want to share or
reference. Hosted via GitHub Pages at
[https://4toddt.github.io/tools/](https://4toddt.github.io/tools/).

## What's here

- [`/fiscal/`](https://4toddt.github.io/tools/fiscal/) — La-Z-Boy 5-4-4
  fiscal calendar (NRF-style). Last Saturday of April year-end, with a 53rd
  week added when the calendar requires it.

## Conventions

- Each tool lives in its own subfolder with an `index.html`.
- Static-only — no build step, no server. Open `index.html` locally or
  let GitHub Pages serve it.
- Keep dependencies minimal; prefer system fonts and inline CSS so the page
  is fast and self-contained.
